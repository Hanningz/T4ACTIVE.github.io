Device Binding Object

Description

This Object defines the link between the EMM and the device it's measuring

Data Models

LWM2M

Object Version: urn:oma:lwm2m:x:25012:1

SHA1 Checksum

917d11c244a7271a054a68b1cf1d47c618ca562f

Resource Fields

  • DeviceName
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    20190 R Yes Mandatory String Name of the device under measurement, e.g. "Wireless Access Point".
  • DeviceID
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    20191 R Yes String Identifier for the device under measurement, e.g. serial number, IP or MAC address
  • DevicePhysLoc
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    20192 R No Mandatory String The physical location of the device, e.g. floor/corridor/room
