Log Control Object

Description

This Object defines the log control for the sensor data.

Data Models

LWM2M

Object Version: urn:oma:lwm2m:x:25011:1

SHA1 Checksum

917d11c244a7271a054a68b1cf1d47c618ca562f

Resource Fields

  • LogClass
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    4010 R Yes Mandatory Integer 0-255 Part of the Logging Event Set: 1/5 (Must be a Single-Instance Resource) Define the Log Event Class: 0: generic (default) 1: system 2: security 3: event 4: trace 5: panic 6: charging [7-99]: reserved [100-255]: vendor specific
  • LogStart
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    4011 E Yes Mandatory String Part of the Logging Event Set: 2/5 Actions: a) Start data collection(DC) b) LogStatus is set to 0 (running) c) DC is emptied(default) or extended according to arg'0' value d) DC Period is set according to arg'1' value Arguments '0': DC emptied(=0[default] or extended(=1) '1': set the DC Period (in sec): if=0 or absent[default] DC is stopped by LogStop action only.
  • LogStop

    4012 E Yes Mandatory String Log Event Set Interface 3/5 Actions: a) Stop data collection(DC) b) LogStatus is set to 1 (stopped) c) DC is kept(default) or emptied according to arg'0' value Argument: '0': DC kept(=0)[default]or emptied(=1)
