Gateway Wireless Interface Object

Description

This Object defines the basic settings for a wireless interface.

Data Models

LWM2M

Object Version: urn:oma:lwm2m:ext:1025:1

SHA1 Checksum

9b5d6f3831af41d19db3f871fe5864635d36c0be

Resource Fields

  • Ifname
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    0 R,W No Mandatory String Name of the Wireless Interface
  • Mode
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    1 R,W No Mandatory String Operation mode of the wireless interface
  • Disabled
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    2 R,W No Optional Boolean When set to 1, wireless interface is disabled
  • SSID
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    3 R,W No Mandatory String The broadcasted SSID
  • MAC Address
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    4 R,W No Optional String MAC Address of the Interface
  • IP Address List
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    5 R,W No Optional String List of IP addresses of the wireless interface
  • Transmission Power
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    6 R,W No Optional Integer dBM Transmission Power
  • Network
    • Resource ID Access Type Multiple Instances Mandatory Type Range or Enumeration Units Description
    7 R,W No Mandatory String Attached Network Interface

lwm2m_object_t * get_object_wireless() // *** MODIFY ***
{
    lwm2m_object_t * objNetIf;
    char ifname[100];
    char *linePtr = NULL;
    FILE *fp;
    ssize_t read;
    ssize_t len;
    uint16_t icount = 0;

    objNetIf = (lwm2m_object_t *)malloc(sizeof(lwm2m_object_t));

    if (NULL != objNetIf)
      {
        int i;
        prv_instance_t * targetP;

        memset(objNetIf, 0, sizeof(lwm2m_object_t));

        objNetIf->objID = 1025; // *** MODIFY: the object ID ***

	// Scan for interfaces currently configured
	systemCmd("uci show wireless | grep ]=wifi-iface > output.txt");
	fp = fopen("output.txt","r");
	if (fp != NULL) {
	  do {
	    read = getline(&linePtr, &len, fp);
	    if (read > 0) {
	      char* pos = linePtr;
	      
	      // Advance pointer until dot or end of read data
	      while (*pos != '@' && pos <= linePtr+read) pos++;

	      if (*pos == '@') { // Was dot found?
		pos++; // Advance pointer over @

		// Advance pointer until equal sign or end of read data
		// copy characters while seeking equal sign
		int i = 0;
		while (*pos != '=' && pos <= linePtr+read) {
		  ifname[i] = *pos;
		  i++;
		  pos++;
		}
		ifname[i] = '\0'; // Terminate string

		// Check if we found equal sign, if so, we have ifname
		if (*pos == '=') {
		  printf("Found wireless interface: >%s<, creating object instance %i\n", ifname, icount);
		  targetP = (prv_instance_t *)malloc(sizeof(prv_instance_t));
		  if (NULL == targetP) return NULL;
		  memset(targetP, 0, sizeof(prv_instance_t));
		  targetP->shortID = icount;
		  icount++;
		  char *tempPtr = malloc(strlen(ifname)+1);
		  strcpy(tempPtr, ifname);
		  targetP->ifname = tempPtr;
		  objNetIf->instanceList = LWM2M_LIST_ADD(objNetIf->instanceList, targetP);
		}	       
	      }
	    }
	    free(linePtr);
	    linePtr = NULL;
	  } while (read > 0);
	}

	/* Create two object instances */
	/*	  for (i=0 ; i < 2 ; i++)
	    {
	      targetP = (prv_instance_t *)malloc(sizeof(prv_instance_t));
	      if (NULL == targetP) return NULL;
	      memset(targetP, 0, sizeof(prv_instance_t));
	      targetP->shortID = i;
	      if (i == 0) targetP->ifname = "radio0"; //@wifi-iface[?]
	      if (i == 1) targetP->ifname = "radio1"; //@wifi-iface[?]
	      objNetIf->instanceList = LWM2M_LIST_ADD(objNetIf->instanceList, targetP);
	    }
	*/

        /*
         * From a single instance object, two more functions are available.
         * - The first one (createFunc) create a new instance and filled it with the provided informations. If an ID is
         *   provided a check is done for verifying his disponibility, or a new one is generated.
         * - The other one (deleteFunc) delete an instance by removing it from the instance list (and freeing the memory
         *   allocated to it)
         */
        objNetIf->readFunc = prv_wireless_read;
        objNetIf->writeFunc = prv_wireless_write;
        objNetIf->createFunc = prv_wireless_create;
        objNetIf->deleteFunc = prv_wireless_delete;
        objNetIf->executeFunc = prv_wireless_exec;
	objNetIf->closeFunc = prv_wireless_close;
    }

    return objNetIf;
}

UCI

Description

OpenWRT gateway native data model for wireless settings and wifi network definition

URI

file://localhost/etc/config/wireless

config

wifi-iface

The system section contains settings that apply to the most basic operation of the system.

Option Fields

  • device
    • Type Required Default Description
    string yes (first device id) Specifies the used wireless adapter.
  • mode
    • Type Required Default Description
    string yes ap Selects the operation mode of the wireless interface
  • disabled
    • Type Required Default Description
    boolean no 0 When set to 1, wireless network is disabled.
  • ssid
    • Type Required Default Description
    string yes OpenWrt The broadcasted SSID of the wireless network.
  • macaddr
    • Type Required Default Description
    MAC address no (hostapd/driver default) Overrides the MAC address used for the wifi interface.
  • network
    • Type Required Default Description
    string yes lan Specifies the network interface to attach the wireless to.
